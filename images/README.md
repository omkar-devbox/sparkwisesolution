## Image Format Guidelines

- **Format**: JPEG for photos, PNG for graphics/icons, SVG for scalable icons
- **Optimization**: Compress images for web (aim for <200KB per image)
- **Alt Text**: All images should have descriptive alt text (already added in HTML)
- **Responsive**: Images should be responsive and work well on all screen sizes
